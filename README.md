# ERP_Proyecto
Prototipo de un ERP con 9 módulos básicos e indispensables para  cada negocio, desde pequeñas empresas hasta mega empresas, el proyecto será desarrollado usando flask y python además de boostrap

## Modulos
``` mermaid
mindmap

    COMPRAS

    "Y MÁS"
    FINANZAS
```
